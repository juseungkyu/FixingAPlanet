## 프로젝트 개요

일반인이 자신만의 행성을 만들어보기 위해서 3D 모델링 툴을 배우고 제작하고 공유하는 과정이 굉장히 복잡하다고 생각되어 이 프로젝트를 만들게 되었다. 

이 프로젝트는 이동, 지형, 색, 해수면, 구름과 같이 기본적으로 제공하는 5개의 툴을 이용해서 행성을 쉽게 제작, 공유 할 수 있다.
